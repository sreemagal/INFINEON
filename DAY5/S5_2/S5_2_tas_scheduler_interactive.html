Time-Aware Shaper (TAS)

IEEE 802.1Qbv · Gate Control List scheduler — gates open and close on a pre-computed nanosecond schedule

Core principle: Each egress queue has a gate. The Gate Control List (GCL) specifies exactly when each gate opens and closes within a repeating cycle period. When a gate is closed, no frame from that queue may leave the switch — regardless of how long it has been waiting. Deterministic, bounded latency is guaranteed because safety-critical streams always get their dedicated window. Requires gPTP clock synchronisation across all switches so gate windows align end-to-end.
