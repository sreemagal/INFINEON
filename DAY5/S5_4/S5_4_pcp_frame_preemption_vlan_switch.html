How a 3-bit PCP field
ends up cutting a frame in half.

PCP does not, by itself, preempt anything. It is a classification value. Inside a VLAN switch it walks down a chain of mappings — Traffic Class, then express/preemptable assignment, then MAC Merge — and only at the very end does the MAC layer decide to interrupt a lower-priority frame mid-flight. This animation walks the chain stage by stage.

▸ Current step

01 · Egress port prepared

The egress port has been configured for frame preemption: TC7 maps to the express MAC, TC1 to the preemptable MAC. The peer has completed verification, so MAC Merge is operationally active.

▸ Standards meaning

802.1Qbu binds traffic-class queues to express or preemptable MAC paths. 802.3br MAC Merge defines the actual mPacket interruption mechanism, which becomes legal only after verification mFrames complete.

▸ Implementation view

A practical switch holds three configuration objects per egress port: the PCP→TC table, the TC→queue mapping, and the queue-to-MAC (express or preemptable) assignment, plus the live MAC Merge state.

The 3-bit Priority Code Point is the first 3 bits of the Tag Control Information. It carries 8 possible values (0–7). The bridge uses it to select an internal Traffic Class. PCP itself never appears on the on-wire fragment delimiter — it is purely a classification input.

▸ Sequence reference

Twelve actions, in order

  1. Port policy is configured. PCP→TC table, TC→queue mapping, and queue→MAC (eMAC or pMAC) assignment are loaded. MAC Merge verification has completed with the peer. verified
  2. Frame L arrives at the ingress port carrying PCP=1 in its VLAN tag. PCP 1
  3. Parser extracts TPID, PCP, DEI, VID from the 802.1Q tag and attaches them as descriptor metadata.
  4. QoS map applies PCP 1 → TC1. Forwarding decides this egress port. TC1 is bound to pMAC.
  5. L is enqueued in the preemptable queue (TC1).
  6. Selector chooses pMAC — no express traffic is pending. L begins MAC-level transmission as a preemptable mPacket. SMD-S0
  7. Frame H arrives at ingress while L is still on the wire. PCP=6. PCP 6
  8. Parser extracts H's tag fields.
  9. QoS map applies PCP 6 → TC7. TC7 is bound to eMAC.
  10. H is enqueued in the express queue. The selector now sees pending express traffic.
  11. MAC Merge preempts L at a valid fragment boundary. L's pMAC state is held; eMAC inserts H. The on-wire delimiter switches from SMD-S* to SMD-E. SMD-E
  12. L resumes as a continuation fragment after H completes. The peer reassembles L from start + continuation. SMD-C*
