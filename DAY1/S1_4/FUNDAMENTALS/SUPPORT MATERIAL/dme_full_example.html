Differential Manchester Encoding — Full Worked Example

Encoding the bit sequence 1 0 1 1 0 0 1 · starting state: HIGH · period width proportional to display

How to read this diagram: Amber circles mark the mandatory mid-bit transitions — present in every period, they provide the clock. Red bars mark start-of-period transitions — present only when the bit is 0, they carry the data. The decode table below shows how the receiver determines each bit by checking whether a start-of-period transition occurred.

Counting shortcut: Rather than tracking the absolute signal level, the receiver simply counts transitions per bit period. One transition → bit = 1. Two transitions → bit = 0. This rule holds regardless of the signal polarity or the starting state, confirming that no initialisation handshake is required between sender and receiver.
